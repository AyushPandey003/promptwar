# GenAI Services Used in the Submission

## Services Utilized
- **Google Gemini API (`gemini-2.5-flash`)**
- **Google Stitch** for UI/design inspiration and interface direction

## Where They Were Used

### Google Gemini API
Used in the **server-side plan generation flow**.

Gemini generates the structured daily cooking plan, including:
- breakfast suggestions
- lunch suggestions
- dinner suggestions
- grocery list
- substitutions
- budget reasoning
- cooking timeline

The generation is based on the user’s:
- day type
- pantry
- budget
- dietary preference
- cooking context and food-story background

### Google Stitch
Used for the **design direction and UI theming** of the application.

Stitch-inspired layouts and screens were used to shape:
- the Herb & Hearth visual theme
- dashboard structure
- meals page layout
- groceries page layout
- soft card-based interface patterns
- spacing, color, and rounded component styling

## Final Summary
The submission uses **Gemini** for intelligent meal-plan generation and **Stitch** for the visual design system and interface inspiration.
